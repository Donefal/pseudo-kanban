# Pseudo Kanban Board
## A very simple implementation of Kanban Board using Bootstrap and Vanilla JS

> *A Javascript assignment project for the **Website Programming** course on UPI's Computer Engineering undergraduate study program*

This project is a form of experimentation of Javascript programming language in the field of websites. Most part of this project is aimed for learning the language itself and to explore the possibility of Bootstrap 5 to use it to make certain website features. The main feature of this project is as follow:
- Create, Read, Update, and Delete (CRUD) feature
- Persistence NoSQL implementation using `localStorage`
- Simplified usual Kanban Board interaction
- Custom HTML tags
